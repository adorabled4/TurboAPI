#  敏感词过滤

### 接口信息

:::info

**敏感词过滤,将text中的敏感词信息变成**, 即便是通过字符分隔也能正确识别到。单次请求text最长为4096个字符**

- 接口状态:  `可用`
- 请求方式: `POST`
- 消耗积分数: `1`
- Tags: `生活服务` `应用开发` `限时免费` 

:::
### 请求地址

```
http://localhost:88api/v3/review/text
```

### 请求参数

**暂无**

### 响应参数

|  参数名称   |  类型  |  描述  |
| :---------: | :----: |:----:|
|    code     |  int   | 状态码  |
|    data     | string | 响应数据 |
|   message   | string | 响应描述 |
