# 成语分析API

### 接口信息

:::info

**输入成语,返回成语的拼音,详细解释,出处,近义词同义词等相关信息**

- 接口状态:  `可用`
- 请求方式: `GET`
- 消耗积分数: `1`
- Tags: `数据智能` `文娱视频` `免费接口` `限时免费` 

:::
### 请求地址

```
https://turboapi.dhx.icu/api/v3/review/text
```

### 请求参数

```json
{`wd`:`string`}
```

### 响应参数

|  参数名称   |  类型  |  描述  |
| :---------: | :----: |:----:|
|    code     |  int   | 状态码  |
|    data     | string | 响应数据 |
|   message   | string | 响应描述 |

### 请求示例

```json
{
    "wd":"一马当先"
}
```
